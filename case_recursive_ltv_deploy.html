GCP VM Deployment (Gunicorn + Nginx + systemd)

This bundle deploys RecursiveLTV on a single GCP VM with: - gunicorn web service managed by systemd - nginx reverse proxy on port 80 - daily sync via either systemd timer (default) or cron

1. Bootstrap host

cd /opt/recursive-ltv/app
bash deploy/scripts/bootstrap_gcp_vm.sh

2. Configure environment

sudo mkdir -p /opt/recursive-ltv/env
sudo cp deploy/env/recursive-ltv.env.example /opt/recursive-ltv/env/recursive-ltv.env
sudo nano /opt/recursive-ltv/env/recursive-ltv.env

Required for credentials panel encryption:

python -c "from cryptography.fernet import Fernet; print(Fernet.generate_key().decode())"

Set the generated value as SECRET_ENCRYPTION_KEY in /opt/recursive-ltv/env/recursive-ltv.env.

3. Install app and migrate DB

bash deploy/scripts/deploy_app.sh

4. Configure nginx + systemd

bash deploy/scripts/install_nginx.sh
bash deploy/scripts/install_systemd.sh

5. Verify services

sudo systemctl status recursive-ltv-web.service
sudo systemctl status recursive-ltv-sync.timer
curl -sS http://127.0.0.1:8000/health

Optional: use cron instead of timer

bash deploy/scripts/install_cron.sh 90

File locations

  • Systemd units: /etc/systemd/system/recursive-ltv-*.service, /etc/systemd/system/recursive-ltv-sync.timer
  • Nginx vhost: /etc/nginx/sites-available/recursive-ltv.conf
  • App env file: /opt/recursive-ltv/env/recursive-ltv.env
  • App root: /opt/recursive-ltv/app
