Cohort maturity is controlled for. Data was snapshotted on 2026-01-15, so cohorts installed within the last N days don't have N-day revenue yet. All ROAS/LTV figures below are computed only from cohorts that are old enough for the selected window (this is what the "coverage %" on each KPI tells you). Switch windows (D1 / D3 / D7 / D14 / D30) to trade off prediction horizon vs. sample coverage. Naive D30 ROAS would mix matured and un-matured cohorts and underestimate performance — don't use that number for softlaunch decisions.
